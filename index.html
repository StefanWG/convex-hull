Graham's Algorithm Visualization

For this assignment, we have created an interactive visualization for Graham's scan algorithm for computing the convex hull of a set of points in the plane. To start, add points to the set by clicking anywhere within the box below. Once you have finished, click the start button. Next, you can either repeatedly press the step buton to illustrate each indivudal step of the algorithm, or you can press the animate button to show each step of the algorithm at fixed time intervals. At each step, the current node that the algorithm is evaluating is outlined in red. Nodes that are apart of the convex hull will be outlined in green and filled in black. Enjoy!
